02 · спроектировано Следующее Реалистичное ядро для nutricore (OpenAI-бот). round-2 + персональная база
1 · Захват

Всё вместе + быстрый выбор

Фото + текст + штрих-код + этикетка как один сигнал.

Быстрый выбор из сохранённых / недавних блюд → поменять количество, минуя анализ.

2 · Разрешение

Ранжирование источников

saved/RAG→ barcode→ name/RAG→ label_ocr→ web→ vision

Текст → структурный JSON → ретрив по персональной + продуктовой базе.

3 · Уверенность

Гейт из трёх скоров

identity portion nutrition
Правила сверки → уверенно: авто-принять · нет: уточнить быстрыми кнопками (порция S/M/L, выбор кандидата).
4 · Черновик

Черновик + обучение

Итоговый черновик → сохранить.

Learning loop: правки пользователя пишутся в персональную базу как подтверждённые алиасы → быстрее в следующий раз.

03 · ориентир North-star «Когда-нибудь» — далеко от текущего OpenAI-бота. из ресёрча · on-prem
1 · Захват

Локально и приватно

Локальный VLM (Qwen2.5-VL / Mistral Small 3.1) как «мозг приёма».

EXIF/GPS-strip, egress по политике, on-prem по умолчанию.

2 · Разрешение

Композитный пайплайн

pgvector RAG USDA / OFF mirror ZXing SAM2 / FoodSeg PaddleOCR

Recognition / matching / nutrient-resolution — как отдельные сервисы.

3 · Уверенность

Скоринг и сверка

Лестница порции (depth / geometry), source-prior скоринг, детерминированные reconciliation-правила.

Полный provenance каждого числа.

4 · Черновик

Интероперабельность

FHIR-экспорт: NutritionIntake · NutritionProduct · Observation · Provenance.

Политики удаления/ретенции, аудит-логи, связь с my-health.

Бейджи источника (реальные строки бота)

📦 по штрих-кодуточно — barcode → OFF
✅ из базыточно — high-tier
🔍 нашли в базепроверь — name → OFF
🏷 с этикеткипроверь — round-2 · label_ocr
🌐 нашли в сетисверь — round-2 · name_web
📷 оценка по фотоlow — vision fallback

Уверенность (цель — три отдельных скора)

identityтот ли продукт опознан
portionнадёжна ли оценка количества
nutritionдостоверны ли итоговые КБЖУ

Принцип: наименьший вопрос, снимающий наибольшую неопределённость — уточнять кнопкой, не гадать молча.
